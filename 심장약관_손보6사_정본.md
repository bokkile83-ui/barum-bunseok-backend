## 1. ★최상위 매핑 규칙 (전 보험사 공통 — 이게 핵심)

**① 묶음 = 구성질환 마스터 행에 동일 금액 각각 기재.** (세로 중복합산 없음, 끝열=가로SUM)

**② 협심증 / 허혈성 칸 분리 (가장 많이 틀리는 곳)**
- 묶음 안의 **I20분 → '협심증' 칸** (허혈성 칸 아님)
- **'허혈성 진단비' 칸 = 허혈성심장질환진단비 '단독담보'일 때만 채운다.**
  → 일반 묶음담보(확대·특정심장 등)에는 **허혈성 칸 절대 안 씀.** I24·25분은 협심증 칸으로 흡수.
  → **예외: '순환계진단비'는 허혈성 칸을 쓴다**(③ 참조).

**③ 묶음 종류 2분류**
| 묶음 종류 | 채우는 칸 |
|---|---|
| **순환계진단비** (심장 전체 커버) | **협심증·심부전·염증·허혈성·부정맥 5개 전부** (+뇌혈관, 약관 코드 세부 무관) |
| **확대/특정심장질환** (I20 포함형) | **협심증·심부전·(부정맥)** — 허혈성 칸 안 씀 |
| 허혈성심장질환진단비 **단독담보** | 허혈성 진단비 칸 |

**④ 급성심근경색은 묶음에 안 넣는다.** 거의 항상 특정Ⅱ/Ⅲ로 분리되어 '급성심근경색' 단독 담보로 들어옴. (묶음과 같이 오는 유일한 경우 = 허혈성진단비 단독담보 I20·24·25+I21~23, 이건 묶음 아님)

**⑤ 빈맥(I47·48)·심근병증(I42·43) = 전 묶음에서 제외/[확인].** 매핑 불가 담보도 누락 금지 → [확인] 큐.

## 3. ★KB '심장질환(특정Ⅰ/Ⅱ)' (김선우 케이스)
명칭상 KB 표준 단계 단정 불가 → **한장보장표(보험사 자체 분류) 우선.**
- 김선우 한장보장표: 특정Ⅰ(700)→허혈성심장질환진단비 / 특정Ⅱ(1,000)→급성심근경색.
- → 매핑: 특정Ⅰ→허혈성 진단비(단독으로 분류됐으므로 허혈성 칸) / 특정Ⅱ→급성심근경색.
- 한장보장표에 분류 없으면 [확인].
