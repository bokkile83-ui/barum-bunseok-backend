## 0. 통신 규칙

결론→근거→실행. 단정체. 위로·감성·장담 금지. 옵션 3개 이내. 서두·반복 금지.
제출 전: 셀프렌더→육안검수→셀프팩폭 먼저. 통과한 것만 제출. 미통과 시 "완료" 금지, 어디가 틀렸는지 자진보고. 모르면 빈칸+[확인], 추측 금지. 안내는 항상 클릭 가능한 직접 링크. 매 답변 끝 핵심 지침 한 줄 재명시.
직접 본 것만 사실. 못 본 것(Railway 런타임 등)은 '직접 못 본다' 명시.

## 1. 핵심 등식

- **등식1** — 한장보장표 = 엑셀 담보값 = PPT 담보값 = 보장진단서 값. (별첨·요약표 이중 대조)
- **등식2** — PPT·진단서는 완성 엑셀만 읽는다. 합계 = 엑셀 데이터셀(C~끝열-1) 직접 합산(=SUM 캐시 의존 금지).
- **등식3** — 슬래시 행(종수술비 1~5종·n대·MRI 트리오)은 합산·최댓값으로 뭉개지 말고 **가로 슬래시 그대로** 엑셀·PPT·진단서 전부 동일 표기.

## 2. 입력·동작

입력 = 사용자가 Adobe Acrobat으로 변환한 .txt 1개 → **엑셀 1개 + PPT 2개(보장분석지·치료비) + 보장진단서 PPT + 보장설명지 PDF** 자동.
PDF 직접 스캔·파싱 금지. 줄단위 직독. 1인 1파일, 새 고객 업로드 시 이전 값 전부 0.
고객명 = txt 파일명 우선, 없으면 내용("○○○ 고객님").
행 삭제·이동·재정렬·폰트변경·새 레이아웃 금지. 추가만. 마스터 열어 값만 채움.

## 3. 추출 엔진

.txt 줄단위 직독. **resolve_kw(결정론적 키워드) 선행 → LLM(Haiku)은 null 채움 전용.** 순서 역전 금지(간병인·암주요치료비 등 확정담보 가로채기 방지).
회사명 3단 복원: 목록 회사열→별첨 제목→상품명. 없으면 "미확인". 발행사를 타계약에 전파 금지.
소스 순서: ①별첨 보장현황(정본) ②세부가입현황(교차) ③한장보장표(검산).
생보 담보 분류 = 보장분석지 좌측 한장보장표 우선(약관 추정 금지), 금액은 별첨 대조.
자동화 천장 ≈ 90%. 깨진 별첨 = [확인] → 신인 수기.
**누락0 원칙: 제외 4종 외 모든 담보 반드시 등장. 매핑되면 해당 행, null이면 [확인] 큐. 증발 금지.**

## 5. 건수·합계

최대 50건. 계약 = 열(C부터). 합계는 항상 맨 끝 열.
각 담보 행 = 계약 가로 SUM(동적, 하드코딩 금지). 미가입 담보행도 0 표기(공란 금지). 보험료 합계 = 가로 SUM, 숫자만.
수술비 1~5종·종수술 2행 + MRI 트리오는 슬래시 합산(숫자 SUM 예외).
**합계는 openpyxl 수식 캐시가 폰에서 공란이 되므로, 데이터셀 직접 합산한 계산값을 기재(등식2).**

---

## 6. 갱신/비갱신 판정 (순서)

① 담보명 '갱신형' 명시 → 갱신
② 만기 9999(종신) → 비갱신
③ 총회차 240 초과 → 비갱신
④ 납입기간 == 보장기간 → 갱신 / 다르면 비갱신
운전자도 동일. 일상배상책임 = 무조건 갱신.

---

## 7. 수술비 1~5종 (슬래시 정본)

상해 종수술비 1행 + 질병 종수술비 1행 = 총 2행. 종별 슬래시(예 3000/100/100/200/400).
다계약 종별 합산 슬래시. 끝열 합계도 종별 슬래시. PPT·진단서 동일.
**6~8종 값이 있으면 (1-8종) 행에 8칸 슬래시.**
**부위·상해/질병 미표기 '수술(1-5종)'·'수술(N종)' → 종수술비공통 → 상해·질병 양쪽 슬래시.**
**'1종~5종'·'1-5종' 범위표기도 종수술비공통으로 처리(v30g 팬텀가드 폐기). 한화 등 1종이 최고액인 상품 정상.**

## 8. 담보 매핑 (B열 담보명 기준 동적, 행번호 하드코딩 금지)

### 8.1 사망
일반사망 = 생명보험사 만기 9999(종신) 표기 사망만 → 일반사망(종신). 종신 시 일반사망+상해사망 1:1.
일반상해사망·재해사망 → 상해사망. 교통상해사망(운전자) → 교통상해사망. 암·특정 사망은 일반사망 미표기.
손보 질병사망 = 80세 칸.

### 8.2 암 (★ 암주요치료비 = 상시 최다 누락 지점, 최우선 점검)
- 암진단비(유사암제외) → 일반암(=PPT 암진단비). **'암진단비(…)' 괄호 유사암 구성은 무시하고 무조건 일반암.**
- 유사암 → 유사암(갑.기.경.제), 환산 ≤2020 ÷10 / ≥2021 ÷5.
- 통합암·통합전이암·전이암진단비 = **통합전이암 행, 대표금액 1개**(부위별·Ⅱ/Ⅲ 다줄 합산 금지). 중대한 암과 별개.
- 표적항암 = 최댓값 1건. 중입자='항암중입자방사선치료비'만(카티 제외). 세기조절/양성자 전용행.
- 하이클래스 = 비급여주요치료비.
- **★ 암주요치료비 (2024~ 대세담보, 변천사 §8.2a).** 우선순위: 하이클래스/비급여 → 하이클래스(암) / 순환계·2대·뇌·심 → 2대주요치료비 / **그 외 '암 + 주요치료/특정치료/진단후 종합병원(상급종합병원) 특정치료(지원금/비)' → 암주요치료비, 대표 1개(max).** 통원·입원일당·재활·MRI·PET·검사·수술은 제외(각 전용행).

### 8.3 뇌혈관·심장
심장 = 담보명 직행, 타칸 전파 금지(2대주요치료비 예외). 허혈성진단→'허혈성 진단비' 행 / 허혈성수술→허혈성수술비 / 심장수술→심장수술비. 라이나 '뇌혈관진단'=실제 뇌출혈→뇌출혈진단비.
**외상성(특정)뇌출혈 → '외상성뇌출혈' 행(마스터 34행) 전용, 뇌출혈진단비에 넣지 마라. PPT 미기재.** 외상성뇌출혈(S06)=뇌혈관진단비 미보장(산정특례only).
2대주요치료비 = 뇌혈관·심장 두 칸 동일 금액 동시. 혈전용해치료비 = 뇌·심 각각 별개.
뇌혈관·허혈성심장 '특정치료비/주요치료비'(암 아님) → 2대주요치료비.

### 8.5 수술비 (★ 과다합산 상시 점검)
- 상해수술비 = 기본만. 병원규모·부위·특정·흉터·복원·**통원·외래** 변형 = 합산 금지 → [확인]. **재해수술비 = 상해수술비 동일 취급.**
- 질병수술비 행 = 담보명 '질병수술비'·'질병입원수술비' 2개만 합산. 변형(특정/부위/Ⅱ/Ⅲ/대수술/종합/일당/**통원/외래/5대**/2~5종)은 합산 금지 → [확인]. **'질병수술비' vs '○○질병수술비'(접두 변형)는 다른 담보, 합치지 마라.**
- **수술비(입원)/(통원)은 둘 중 1개만.** (통원)형은 base 행 유입 차단([확인]). 상해·질병·재해 동일.
- 1XX대(116·119·120·123)→n대수술비 대표 1행. **[미완] n대수술비 계약별 슬래시 전개는 미구현 — 현재 대표 1건.**
- 5대기관(관혈)→관혈 / (비관혈)→비관혈. 뇌혈관수술→뇌혈관수술비. 허혈→허혈성수술비. 창상봉합→창상봉합술(대표 최댓값 1건).

### 8.6 운전자
벌금(대인)·벌금(대인대물 미표기)→대인(단독 1건, 과실치사·업무과실 변형 이중합산 금지→[확인]). 벌금(대물)→대물. 화재벌금 제외. 처리지원금(중상해)→합의금, (6주미만)→6주미만, 변호사→변호사. 합의금 최대 2.5억, 초과 [확인].
자부상 = 14급(경상) 포함 밴드만(예 12~14급/4~14급). 1~3·1~7급 제외. 자동차 한정. 합산·최댓값 폐기 = 밴드 금액 1건. 캡 80만.

### 8.7 골절·일당·화상
골절 '치아포함'/단독→골절(치아파절포함), '치아제외/파절제외' 명시만→골절(치아파절제외). 두 행 안 섞음.
골절(치아파절 포함/제외)·화상진단비 100만↑ = 등급별→합산·기재 안 함([확인], 100만 미만만). 5대골절진단비·중증화상진단비는 별도 행이라 미적용.
질병일당='질병일당'·'질병입원일당'만. 상해일당='상해일당'·'상해입원일당'만. 변형(종합병원·수술·요양 일당)=별도/[확인].
간호간병통합서비스 입원일당→간호통합병동(7만 등 실값). 간병인사용 질병/상해입원일당(요양병원 제외)→간병인 대표 1개. 요양병원 포함형=미기재.
중증·부식·심재성 화상→중증화상진단비. 화상진단비→화상진단비. 골절부목·반깁스→반깁스.

### 8.8 실손 (세대 자동판별)
- 입원 한도 3,000=구형실손 유지 / 그 외 → 5,000 고정. 통원 20~25만·약값 5~10만(표 보고 표기).
- 실손·간병인·일상배상책임 값 = 갱신 무관 항상 파랑.
- MRI/도수치료/비급여주사 = 'MRI/도수치료/비급여주사' 한 행 슬래시(대표 최댓값, 중복합산 금지).
- **세대 판정: 상품코드(YYMM) 우선 → 없으면 가입일.** 경계: 1세대~2009.07 / 2세대 2009.08~2017.03 / 3세대 2017.04~2021.06 / 4세대 2021.07~. 헤더·PPT에 'N세대 실손' 기재. 15년 재가입 시작 = 2013.04.
