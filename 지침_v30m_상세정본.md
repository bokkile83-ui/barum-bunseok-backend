## 7. 수술비 — 계열·종수·결정 트리

### 7.1 세 계열 (범위·금액 반대)
- **질병/상해수술비** = 기본, 포괄주의(면책 외 전부), 관혈/비관혈 구분 없음, 금액 적음(30~50만).
- **N대(112·116·119·120대 등) 수술비** = 열거주의, 구분 없음, 혈관질환 고액이나 범위 좁음.
- **종수술비(1종~N종)** = 종별 차등 정액(1종 백내장 20만 ~ 5종 개흉 1,000만). 종 클수록 금액↑·발생↓.
- 범위: N대 < 종수술비 < 질병/상해수술비. 금액: 반대.

### 7.2 1-5종이 가장 유리 (설명서 권유 근거)
- **1-5종** = KCD 기준·관혈/비관혈 구분·**비급여 보장**·청구 단순·동일질병 1년 1회 최고종. → 가장 유리.
- **1-7/1-8/1-9종** = ADRG 기준·**비급여 미보장**·넓어 보이나 실질 좁음·청구 복잡·보험료 높음·고객 불리.
- **설명서 규칙:** 1-5종 외(1-7/8/9·N대만) 가입 고객 → 단점(비급여 미보장·청구 불리) 설명 + 1-5종 권유.

### 7.3 종수술비 기재 방식
- 상해 종수술비 1행 + 질병 종수술비 1행 = 총 2행. 종별 슬래시(예 3000/100/100/200/400).
- 다계약 = 종별 합산 슬래시. 끝열 합계도 종별 슬래시. PPT·진단서 동일.
- 6~8종 값 있으면 (1-8종) 행에 8칸 슬래시.
- 부위·상해/질병 미표기 '수술(1-5종)'·'수술(N종)'·'1종~5종'·'1-5종' 범위표기 → 종수술비공통(상해·질병 양쪽).

### 7.4 수술비 결정 트리 (담보명 보고 순서대로)
1. 담보명에 '수술' 있음 → **아래 규칙만으로 판정. Haiku 안 씀.** 못 잡으면 [확인].
2. '1종~5종'·'N종'·종번호 표기 → 종수술비(상해/질병).
3. '10~150대'(10·20·116·119·120·123대…) → **n대수술비** 행. 끝열 = 계약별 값 **가로 슬래시**(예 100/500). 합산·최댓값 금지.
4. '5대기관(관혈)' → 5대기관 관혈 / '(비관혈)' → 비관혈. (5대 = 단독 자릿수라 n대 아님)
5. '허혈성수술' → 허혈성수술비 / '심장수술' → 심장수술비 / '뇌혈관수술' → 뇌혈관수술비.
6. '재해수술비' → **상해수술비** 동일 취급.
7. 순수 '질병수술비'·'질병입원수술비' 딱 2개만 → 질병수술비 행. 순수 '상해수술비' → 상해수술비 행.
8. **접두·변형은 전부 별개, 합산 금지 → [확인]:** 특정/부위/Ⅱ/Ⅲ/종합병원/**통원·외래**/5대/20대/○○질병수술비. (예 "20대질병수술비" ≠ "질병수술비". "특정질병수술비" ≠ "질병수술비".)
9. '수술비(입원)'과 '수술비(통원)' 동시 존재 → 둘 중 1개만. (통원)형은 base 유입 차단([확인]).
10. 창상봉합 → 창상봉합술(대표 최댓값 1건).

- 단독: 협심증→협심증 / 허혈(성)→허혈성 진단비 / 부정맥→부정맥 / 심부전→심부전 / 빈맥→빈맥 / 급성심근경색→급성심근경색 / 심장질환 특정Ⅱ→급성심근경색. 심근병증(43)·심장판막(44) = 단독이면 해당 행, 묶음 구성분 아님.

#### 8.3.2 치료비 계열 — 범위 4종 (설명서 근거)
회사별 명칭(2대주요치료비/신특정순환계/뇌혈관·허혈성주요치료비/순환계생활치료비 등) 전부 BARUM '2대 주요치료비' 행(뇌+심 두 칸 동액). 설명서에서는 아래 범위 구분:
- **심장:** 급성심근경색(I21~23,9%) ⊂ 허혈성심장질환(+협심증 I20·24·25,50%) ⊂ 심장특정질환(+판막·염증·부정맥·심근병,41%).
- **뇌:** 뇌출혈(I60~62,9.3%) ⊂ 뇌졸중(+뇌경색 I63·65·66,60.8%) ⊂ 뇌혈관질환(+기타 I64·67·68·69,100%).
- **담보 넓이:** 허혈성(가장 좁음) < 2대(뇌+심) < 순환계(2대 + 동맥류 I71·72·정맥류·류마티스 I00~09·식도정맥류 I85, 가장 넓음).
- **산정특례 = 별개 축(진단 기반).** [뇌] I60~69 + Q28.0~28.3 + S06 / [심] 심장 I00~I52 대부분 + 대동맥 I70~71 + 선천심장 Q20~25 + M31.4·A39.5·B37.6 + 흉부손상 S25~26. 뇌·심 각각(마스터 35·45).
- 설명서 P5 '보장범위 안내'에 이 매트릭스 구현 완료(§12).
