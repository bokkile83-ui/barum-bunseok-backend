## 사용법
```
# 엑셀 채움 (let PDF → 엑셀)
python3 fill_xls.py <let.pdf>

# PPT 채움 (완성엑셀 → PPT)
python3 fill_ppt_from_xlsx.py 보장진단_고객.xlsx 고객명
```

## 핵심 등식
PDF 한장보장표 = 엑셀 담보값 = PPT 담보값

## 드리프트 차단 6규칙
1. 재생성 금지 — 빈 마스터를 열어 값만 끼워넣는다.
2. 추가만 — 없는 담보는 그룹 안에 행/박스 삽입. 삭제·이동·폰트변경 금지.
3. 엑셀=원천, PPT는 완성 엑셀에서만(PDF→엑셀→PPT).
4. PPT 채움은 기존 run .text 교체만. clear 금지.
5. 구분셀 연두 / 갱신=파랑·비갱신=검정·미가입=빈칸 / 1~5종 ( / / / / ) 골격.
6. 모르면 빈칸+[확인], 추측 금지.
