## ① 상품명 매칭 실패 (근본 원인)
```python
'CI보험' in "무배당교보큰사랑 CI 보험"   →  False
```
`CI` 와 `보험` 사이 공백 때문에 `_is_ci` 가 처음부터 False.
CI 로직 전체가 진입조차 못 했다.

→ 수정: `_isci_prod()` 헬퍼 신설. 공백·전각 제거 후 판정. 5곳 전부 교체.

## ② 주계약 수집 실패
```python
re.match(r'^\s*주계약\s+([\d,]+)\s*$', line)   →  0건
```
`pdftotext -layout` 은 한 줄에 담보 2~3개를 붙인다.
실제: `' 주계약   4,000    주계약   3,200    남'` → `$` 앵커 불가능.

→ 수정: `re.finditer(r'(?<![_가-힣])주계약\s+([\d,]{3,})', line)`
   `일반사망_주계약`·`주계약_주계약`(동양생명) 은 lookbehind 로 차단.

## ③ CI추가보장특약 수집 실패
실제 표기는 `CI 추가보장특약` (공백). 줄 중간에 위치.

→ 수정: `re.finditer(r'CI\s*추가보장특약\s+([\d,]+)', line)`
