# 오늘 고친 것 전부

## ① 선지급률 39% → 80%  (coverage_benchmark.py · 배포 완료)
`중대한CI적용` 끝열 = CI 잔여 + 비CI 계약 일반사망 전부 → 오염.
```
3,200 / (3,200 + 5,000) = 39%   ← 끝열
3,200 / (3,200 +   800) = 80%   ← CI 계약 열
```
→ CI 계약 열에서 직접 읽는다. 값은 50% 또는 80%로 스냅.

## ② CI 3-state 미작동  (main.py · 배포 완료)
```python
'CI보험' in "무배당교보큰사랑 CI 보험"   →  False
```
`CI` 와 `보험` 사이 공백 때문에 CI 로직이 진입조차 못 했다.
주계약·CI추가보장특약 정규식도 `pdftotext -layout` 다열 레이아웃에 안 맞았다.
→ `_isci_prod()` 공백 무시 판정 + `re.finditer` 로 교체.
