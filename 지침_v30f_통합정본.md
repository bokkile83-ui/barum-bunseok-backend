확인 없이 단정 금지 — 직접 본 것만 사실. GitHub: api 403이면 raw 재시도 필수. Railway 런타임 로그는 못 본다고 명시.
모르면 빈칸+[확인], 추측 금지. 폰 작업 = 한 번에 한 단계, 통째 복붙 블록.

## 1. 연계 구조 — 등식 사슬 (v30f 확정)
완성 엑셀 = 단일 정본. PPT·보장설명지 = 읽기 전용 뷰. 판단(매핑·대표·캡·[확인])은 txt→엑셀 단계에서만.
등식1: 원문(별첨) ↔ 엑셀 — 누락0(증발 금지, 못 잡으면 [확인] 큐).
등식2: PPT ↔ 엑셀 — 표기 총액의 정본 = 완성 엑셀 끝열. raw 분할합(갱신+비갱신)이 끝열과 다르면(대표·MAX·캡 적용분) 끝열 값을 우세 성분측 단색 표기. 분할색은 비율 용도로만.
등식3: 보장설명지 ↔ 엑셀 — 모든 표기값은 엑셀 행값과 1:1. 심장 묶음 전개(1건→N행)는 [근거] 시트를 읽어 보유합에 1회만 카운트. 혈전용해치료비(심장)는 설명지에서 심장 카테고리로 재배치.
값 주입 원칙: dambo(파싱) 변환이 정본 — 엑셀 셀 직접기재는 PPT 분할합을 우회해 등식1을 깬다(일당·CI 사망 2회 사고). 예외는 CI 본체·급성심근 초과분 등 재매핑 회피 목적만.

## 2~4. 입력·추출·제외
.txt 1개 → 엑셀+PPT+설명지. 줄단위 직독, PDF 파싱 금지.
추출: 같은줄 우선 + 분리줄 페어링(코드[x]/이름랩/금액뭉치 FIFO). 금액 미확보 = 0+[확인].
제외 4종만: 실효·미납해지·농업인('농업인' 표기만)·자동차보험.

## 5. 합계 (v29u~v30d)
전 담보행 = 동적 수식(하드코딩 금지): 기본 =SUM / 입원 =MIN(SUM,5000) / 자부상 =MIN(SUM,80) / 간병인·중입자치료비·120대수술비 =MAX(전 계약 대표 1건) / 간호통합 =IF(>0,7).
recalc(LibreOffice) 실패 시 inject_sum_cache가 캐시값을 XML <v>로 주입(Railway엔 LibreOffice 없음 — 확정 사실). 수식 유지 + 폰·설명지 값 표시.
슬래시 행: 1-5종 5칸 / 1-8종 8칸 / MRI트리오 3칸 — 항목별 합산 슬래시, SUM 예외.

## 6. 갱신 판정
담보명 "갱신형"→갱신(엑셀 값색·PPT 분할합 동일 기준) → 만기9999→비갱신 → 총회차240초과→비갱신 → 납입==보장→갱신.
일상배상책임 무조건 갱신. 실손·간병인·간병인지원일당·일배책 값 = 항상 파랑.

## 7. 종수술비
1-5종 = 상해·질병 각 1행 5칸 슬래시. 1-8종 = 그 밑 전용 2행 — 6~8종 값 존재 계약은 8칸 슬래시.
1XX대(111·112·116·119·120·123·124·130…) = n대수술비 → 마스터 '120대수술비' 행(std↔행명 별칭), 전 계약 통틀어 최댓값 1건.
5대주요기관(관혈)→관혈 행 / (비관혈)→비관혈 행.

## 8. 담보 매핑 (확정 누적)
8.1 사망: 종신(9999) 주계약 = 일반사망+상해사망 1:1('[N] 주계약_주계약' 2줄 분리 포함, CI 포함). 특정사망(암·재해·교통) 일반사망 금지. 손보 질병사망=80세 칸.
CI 사망 양쪽 합산: 중대한CI적용 행 = CI 잔여액(사망−본체) + 비CI 계약 일반사망 동일액. PPT 종신 = 값+CI적용합(노랑).
8.2 암: 암(소액암제외/유사암제외)→일반암(부정어 처리). 표적=최댓값 1건. 항암방사선·약물 = 대표 1건, 특정부위 한정형(생식기·전립선·유방 등)=[확인]. 중입자 = 전 계약 최댓값 1건. 통합암·통합갑상선·통합전이암 = 대표금액 1개(부위별 변형 합산 금지). 갑상선암(통합·초기·중증)=유사암 행. 전이암진단비=통합전이암 전용행(19행). 암진단후 치료지원금(연간밴드)=진단비 아님→[확인]. 유사암 주요치료비·양성종양수술→[확인].
암주요치료비 우선순위: 담보명 '암주요치료' 명시 > 비급여 수식어(→암주요치료비, 대표 1건) / 순수 비급여·하이클래스→하이클래스(암) / 순환계·2대→2대주요치료비. 양성자·세기조절→해당 행 + PPT 암 박스 '양성자.세기' 칸.
8.3 심장(6사 + v30 재점검): 허혈성·허혈심장질환진단→허혈성 진단비(구 협심증 폐기). 허혈수술→허혈성수술비. 빈맥 행 있음(7/1 신설 — "빈맥 무행" 폐기). 질환별 입원일당(허혈·뇌혈관·부정맥 등)=진단비 금지→[확인].
묶음: KB 확대심장/한화 심혈관특정Ⅰ·Ⅱ/메리츠 확대심장 = 협심증·부정맥·심부전·빈맥 / KB 심장질환(특정Ⅰ)Ⅲ = 협심증·심부전·빈맥·염증 4행, 특정Ⅱ=급성심근경색 / 현대 특정Ⅰ(I49제외)=협심증·심부전·빈맥, I49=부정맥, 중증Ⅱ=급성심근경색(회사명으로 한화와 구분) / DB 4종=협심증·심부전·빈맥, 3종=염증·부정맥, 5종(중증)=급성심근경색+뇌졸중 / 삼성 특정순환계=협심증·심부전·빈맥·염증·허혈성·부정맥+뇌혈관진단비. 심장 기재는 전건 [근거] 시트에 원담보명→행 기록.
8.4 CI: 중대한 3종=본체(80/50%). CI추가보장특약=최대 1건→급성심근경색(초과분), 잔여 [확인]. PPT 급성심근 칸=중대한(노랑)+일반.
8.5 수술: 기본만(질병수술비 변형 검사는 접두 상품수식어 제거 후). 5대골절진단→5대골절진단비(부분일치 선순위).
8.6 운전자: 자부상=12~14급 대표만·최대 80캡·가족동승 제외([확인])·1~3/1~5/1~7급 제외. 벌금 대인 단독. 합의금 2.5억 캡.
8.7 일당·간병: 화이트리스트 = 질병일당·질병입원일당 / 상해일당·상해입원일당 / 밴드형 입원비(1일이상·180일) / base 질병입원비·상해입원비(AIG류, 변형·질환한정 [확인]) / 재해=상해 동일. 생보 무수식어 '입원특약'=질병일당+상해일당 두 행 동일 기재. 간병인지원일당 전용행. 상해수술일당 전용행. 고도후유장해=80%후유장해.
8.8 실손: 입원 5,000 고정. 값 우선순위 ①별첨 명시값(외래의료비=통원값) ②구형(입원3000) 10·5 ③손보 25·5/생보 20·10 ④4세대 20·0. 세대=상품명 연도코드(YYMM·YY.MM 점형식 포함) 우선. MRI/도수치료/비급여주사 = 약값 아래 전용 1행 3칸 슬래시(대표 최댓값·항상 파랑·PPT 자동 분해). 세대 판정 근거는 [근거] 시트 자동 기록.

## 9. 마스터 (99행 정본)

## 10~11. 색·선·PPT
