0. 통신 규칙

등식2: PPT·리포트는 완성 엑셀만 읽음(데이터셀 직접 합산, SUM 캐시 의존 금지).

2. 입력·동작
입력 = Adobe Acrobat 변환 .txt 1개 → 엑셀 1개 + PPT 2개(보장분석지·치료비) + 리포트 PDF. 줄단위 직독. PDF 직접 스캔 금지. 1인 1파일, 새 고객 시 이전값 0. 고객명=txt 파일명 우선. 행 삭제·이동·재정렬 금지(추가만).

4. 제외 보험 (이 4종만 제외)

5. 건수·합계

6. 갱신/비갱신 (순서)
담보명 "갱신형" 명시→갱신 / 만기9999(종신)→비갱신 / 총회차240 초과→비갱신 / 납입기간==보장기간→갱신, 다르면 비갱신. 운전자 동일. 일상배상책임=무조건 갱신(ALL).

7. 수술비 1~5종 (슬래시)
