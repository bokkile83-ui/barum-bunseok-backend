# 바름 보장분석 — 통합 정본 지침 (2026.06.15)

> 이 파일 하나가 정본이다. 기존 .md 4개(법칙_100조·법칙_v2·정본폼·오늘변경)는 폐기·대체한다.

## 0. 통신 규칙
결론→근거→실행. 단정체. 위로·감성 금지. 옵션 3개 이내. 서두·반복 금지.
제출 전: 셀프렌더→육안검수→셀프팩폭 먼저. 통과한 것만 제출. 미통과 시 "완료" 금지, 어디가 틀렸는지 자진보고. 모르면 빈칸+[확인], 추측 금지.

## 1. 핵심 등식
PDF 한장보장표 = 엑셀 담보값 = PPT 담보값. (등식1: 별첨·요약표 이중대조 / 등식2: PPT는 완성 엑셀만 읽음)

## 2. 입력·동작
- 입력 = 보장분석지 PDF 1개 → 엑셀 1개 + PPT 1개 자동.
- 1인 1파일. 새 고객 업로드 시 이전 값 전부 zero화.
- 고객명 = PDF 파일명 우선, 없으면 내용("○○○ 고객님").
- 행 삭제·이동·재정렬·폰트변경·새 레이아웃 발명 금지. **추가만.**
- 폼은 파일(.xlsx/.pptx)로 들고 다닌다. 글로 재현 금지. 마스터를 열어 값만 채움.

## 3. ★ 보험 건수·합계 (정정 5 — 최우선)
- **보험은 최대 50건까지 가입 인지.** 계약 = 열(C부터 우측 확장).
- **합계는 항상 표의 맨 끝 열(끝 라인)에 존재.**
- **각 담보 행을 계약 가로로 합산 = 끝열 SUM.** SUM 범위는 계약 수만큼 동적 확장(최대 C~50번째 계약열). 하드코딩 금지.
- **보험료 합계 = 가입된 전 계약 보험료의 가로 SUM.** 숫자만 표기(원·한글 금지).
- 법칙22조: 끝열은 SUM 수식 유지. 하드코딩 숫자 대체 금지(단 수술비 1~5종 2행은 §6 예외).

## 4. ★ 제외 보험 (정정 2)
**제외는 다음 4가지뿐:** ①실효계약 ②미납해지 ③농업인NH안전보험 ④자동차보험.
그 외 전부 보장분석 포함. **NH농협생명·NH농협손보 = 일반보험으로 포함.** 치아보험도 포함.

## 5. 추출 엔진
PyMuPDF(fitz) 단독. poppler 금지. 별첨 직독 줄단위. 회사명 3단 복원(목록 회사열→별첨 제목→상품명), 없으면 "미확인". 이미지형 별첨은 Claude Vision 또는 Adobe 텍스트추출본 사용.
데이터 소스 순서: ①별첨 보험서비스별 보장현황(정본) ②세부가입현황(교차검증) ③한장보장표(검산용).

## 6. ★ 수술비 1~5종 (정정 1 — 슬래시 유지)
- **슬래시 표기가 정본이다. 슬래시 금지 아님.**
- `상해 종수술비(1-5종)` 1행 + `질병 종수술비(1-5종)` 1행 (각 1행, 총 2행).
- 종별 슬래시 표기: 예 `20/50/100/500/1000`. 종별 택1, 빈칸 금지.
- 다계약 시 종별 합산 슬래시: 예 상해 롯데 10/30/100/200/400 + 현대 10/50/100/500/1000 = `20/80/200/700/1400`.
- 끝열 합계 = 종별 합산 슬래시(이 2행은 §3 숫자 SUM 예외). PPT도 동일.
- 상해수술비 = 기본만(병원규모 가산 합산 금지). n대수술비 = 대표 1행, 1~5종과 혼용 금지.

## 7. ★ 갱신/비갱신 판정 (정정 3 — 운전자 규칙 폐기)
판정 순서:
1. 담보명 "갱신형" 명시 → 갱신.
2. 만기 9999(종신) → 비갱신.
3. 총회차 240 초과 → 비갱신.
4. **납입기간 == 보장기간(가입일~만기일) 동일 → 갱신 / 다르면 비갱신.**
- **운전자도 위 4번 규칙 그대로 적용.** "운전자→무조건 비갱신" 규칙 폐기.
- "비갱신암" 같은 단어에 안 속음(날짜 계산 우선).

## 8. 사망·암·뇌·심장
- 9999(생보)=종신 → 일반사망(종신) + 상해사망 1:1. 손보 질병사망 = 80세 칸.
- 일반암: 갱신/비갱신 행 분리. 유사암: ≤2020 ÷10, ≥2021 ÷5. 표적항암=최댓값 1건. 중입자='항암중입자방사선치료비'만(카티 제외). 하이클래스=비급여주요치료비.
- 라이나 '뇌혈관진단'=실제 뇌출혈 → 뇌출혈진단비 칸으로 교정.
- 심장 = 담보명 직행(허혈성→허혈성칸, 전파 금지). 허혈성은 수술비 블록의 '허혈성수술비' 행 사용(심장 블록에 허혈성진단 행 추가 안 함 — §9).

## 8-1. ★ 운전자 매핑 (정정 — 처리지원금=합의금/6주, 벌금=대인/대물)
- 교통사고벌금(대인) → **대인**
- 교통사고벌금(대물) → **대물**
- 교통사고처리지원금(중상해 포함) → **합의금**
- 교통사고처리지원금(6주미만 진단) → **6주미만**
- 변호사비 / 변호사선임비용 → **변호사**
- 자동차부상보장 / 자동차사고부상보장(등급 무관, 운전자) → **자부상**
- 판정 순서: 6주 → 합의금(처리지원금) → 벌금(대인/대물) → 변호사 → 자부상.
- 보험료 납입면제·일시납입지원 = 비담보 → 매핑 안 함(자부상 등으로 흡수 금지).
- **'교통사고처리지원금→대인' 옛 규칙 폐기.** 운전자상해보험은 §4 자동차'보험'이 아니므로 포함.
- **벌금담보(대인/대물 미표기) = 대인.** 벌금(대물) 명시만 대물. ('벌금→대물' 기본 폐기)
- **자부상 = 자동차부상치료비/위로금.** 등급(급수)별 기재는 **12~14급(경상) 기준으로만 합산**이 원칙. (12~14급 금액이 별첨에 없으면 [확인] 후 신인 수기. ※ 최댓값 규칙 폐기)

## 8-2. ★ 상해수술비·화상 매핑 (정정)
- 상해수술비 = **기본 '상해수술비'만.** 병원규모 가산(상급종합병원·종합병원)·부위/특정 변형(흉터복원·외모특정·척추·3대하지관절·안면 등)은 **합산 금지 → [확인]**. (§6 병원규모 가산 합산 금지의 구체화)
- 중대한화상 및 부식진단비 → **중증화상진단비**.
- 화상진단비 → 화상 구분 **'화상진단비'** 칸.

## 8-3. ★ 2대 주요치료비 매핑 (정정 — 신특정순환계)
- 신특정순환계(질환) 주요치료비 / 2대 주요치료비 계열 → **2대 주요치료비**.
- **뇌혈관·심장 두 칸 모두 동일 금액 기재**(2대 = 뇌혈관+허혈성심장 통합 담보).

## 8-4. ★ 암 진단·주요치료비 매핑 (정정)
- 암진단비(유사암제외) → **일반암**(엑셀 '일반암' 행 = PPT '암진단비'). '유사암제외'는 유사암을 뺀 일반 암진단이므로 일반암.
- 유사암진단비(Ⅲ 등 변형 포함) → **유사암(갑.기.경.제)**. (옛 버그: DMAP '암진단비' substring이 '유사암진단비'를 일반암으로 가로채던 것 폐기)
- 암(유사암제외)주요치료비 → **암주요치료비**(신규 행). PPT는 **상급병원 암주요치료비(TextBox 57 p0)** 칸.
- 하이클래스 주요치료비 = 비급여 주요치료비 → **하이클래스(암)**. PPT는 **상급병원 하이클래스(TextBox 57 p1)** 칸.
- 매핑 우선순위(암 주요치료비): 하이클래스/비급여 → 하이클래스(암) / 순환계·2대 → 2대 주요치료비 / 그 외 암+주요치료 → 암주요치료비.

## 8-5. ★ 수술비 매핑 (정정)
- 질병수술비 / 질병입원수술비 → **질병수술비**(기본만). 특정·부위·관절 등 그 외 변형은 **합산 금지 → [확인]**. (상해수술비와 동일 원칙, §6)
- 116대·119대·120대·123대(1XX대) 수술비 → **n대수술비**(대표 1행). **여러 건이면 가장 큰 1건만**(합산 금지). 기존 '120대수술비' 행을 'n대수술비'로 통합.
- 5대기관수술비(관혈) → **5대기관 수술비 관혈**, 5대기관수술비(비관혈) → **5대기관 수술비 비관혈**. 관혈/비관혈 표기대로 각 칸 분리 기재.
- 수술비 블록 내 **질병수술비 행 위에 굵은 구분선** — 상해 수술(위) / 질병 수술(아래) 시각 구분.

## 10. 색 규칙

- 정본 빈 마스터 열어 run-injection으로 값만 교체. **tf.clear() 절대 금지.**
- 도형·위치·색·라벨·폰트 고정. 텍스트박스 이름으로 값만.
- 금액 = '천만/억/만' 한글 단위(예: 1억7,100만). 미가입=비움. 도형 밖 글자 튀어나옴 금지(word_wrap off).
- PPT 값 출처 = 완성 엑셀 끝열(등식2). 매핑 안 되는 값은 비우고 메모.
- B열 담보명 기반 동적 매핑(행번호 하드코딩 금지).
- **PPT 합계 = 완성 엑셀 데이터셀(C~끝열-1) 직접 합산.** 끝열 =SUM() 캐시(LibreOffice 재계산)에 의존하지 않음 → 재계산 실패해도 **항상 PPT=엑셀**.
