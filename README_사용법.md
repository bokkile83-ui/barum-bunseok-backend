# 사용법

## 파일
- `보장설명서_이종수님_편집가능.pptx` — 10슬라이드. 텍스트 도형 321개, 이미지 10개(4p 게이지)
- `보장설명서_김규훈님_편집가능.pptx` — 동일
- `보장진단_이종수님_보정.xlsx` — 확정 오류 10건 보정. 합계 수식 보존
- `보장진단_김규훈님_보정.xlsx` — 확정 오류 6건 보정
- `report_pptx_shapes.py` — 생성기. `python3 report_pptx_shapes.py <보정.xlsx> <출력.pptx>`
- `fix_excel.py` — 엑셀 보정 스크립트
- `확인큐_20260708.md` — 손대지 않은 13건 + 코드 수정 지점

## 편집 규칙
- 값 텍스트는 자유롭게 수정 가능 (321개 전부)
- 도형 위치·크기·색 변경 금지
- **4p 게이지: % 숫자만 고치면 링(그림)과 어긋난다.** 값 변경 시 재생성할 것
- 폰트 = 맑은 고딕 (latin + ea 둘 다 지정. 윈도우 기본이라 안 깨짐)
- 수동 보정분은 재생성 시 소실 → 엑셀에 반영되지 않은 값은 임시본으로만 사용

## 등식
- 등식1: 한장보장표 = 엑셀 끝열 = PPT  ※ 골절진단비에서 구조적 불일치 발견, 확인큐 참조
- 등식2: PPT는 완성 엑셀만 읽는다 (데이터셀 C~끝열-1 직접 합산)
